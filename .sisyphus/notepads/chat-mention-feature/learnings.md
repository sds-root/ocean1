# Learnings & Wisdom

## Project Structure
- shadcn/ui components are in `apps/web/src/components/ui/`
- Mock data in `apps/web/src/lib/mock-data.ts`
- Chat route in `apps/web/src/routes/chat.tsx`

## Conventions
- Use `bun` for package management
- Run commands in `apps/web` for frontend tasks
- Use `cn()` for class merging
